# Salary Prediction using Linear Regression

## Overview
This project predicts employee salaries based on years of experience using a Linear Regression model.

## Dataset
- Salary Data.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Machine Learning Model
- Linear Regression

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Training using Linear Regression
5. Model Evaluation
6. Salary Prediction

## Results
The Linear Regression model learns the relationship between years of experience and salary, enabling salary predictions for new employees.
